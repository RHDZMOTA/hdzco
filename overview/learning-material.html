Books

  • Essential Scala - This book is great for junior/mid developers willing to learn scala.
  • Functional Programming for Mortals with Scalaz - This is a go-to resource for starting FP with Scala. Good understanding of the Scala Programming Language is required.

Youtube videos

Regarding functional programming:

  • Scala Monads: Declutter Your Code with Monadic Design - In this talk Dan Rosen guides the audience via practical examples to an understanding of the monadic design pattern.
  • Options in Futures, how to unsuck them - This talk by Erik Bakker @eamelink at Scala Days 2015 illustrates an intuitive approach to monad transformers.
  • Getting Work Done with the Eff Monad in Scala - Great introduction to the effect monad by Ben Hutchison. Required: monad transformers.
  • The Eff monad, one monad to rule them all - An in-depth explaination of the Eff Monad by Eric Torreborre @etorreborre at Scalar Conf 2016.
